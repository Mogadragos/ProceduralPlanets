## Génération procédurale de système solaire

Pour la création de nos planètes, nous nous sommes basés sur le tutoriel vidéo de Sebastian Lague : https://www.youtube.com/watch?v=QN39W020LqU&list=PLFt_AvWsXl0cONs3T0By4puYy6GM22ko8.

Nous avons ensuite modifié ce système pour générer plusieurs planètes aux propriétés aléatoires dans un système complet.

En particulier, les règles suivantes influent sur l'aléatoire :
- Plus une planète est proche du soleil, plus elle est "rocheuse" ;
- Plus elle est proche du soleil, plus elle est teintée en rouge (couleurs plus chaudes) ;
- Plus elle est éloignée du soleil, plus elle est teintée en bleu (couleurs plus froides) ;
- Si elle est à une distance intermédiaire, la teinte n'influence pas sa couleur finale ;
